Change map
  changelevel {mapname}
    builtin maps are start, e1m1, e1m2, ..., e1m8

To play singleplayer (coop)
  deathmatch 0
  coop 1
  skill 3 (3 is hardest, 0 is easiest)
  changelevel e1m1

Deathmatch (player vs player)
  deathmatch 1
  coop 0
  changelevel e1m7

TO CONNECT:
  go to multiplayer menu on other computer and type in one of
  the addresses on the bottom of the screen
